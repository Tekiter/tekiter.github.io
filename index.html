Intro

웹 개발을 공부해가고 있는 학생입니다. 여러가지 토이 프로젝트들을 기획하고 제작해 본 경험이 있습니다. 어떻게 하면 사람이 이해하기 쉽고, 변화에 유연한 코드를 작성할 수 있을까에 대해 관심이 많습니다.

능동적으로 배우는 것에 익숙하고, 새로운 것을 알아가는 것을 즐거워 합니다. 그동안 만들고 싶은 것을 먼저 정하고, 그것을 만드는 데 필요한 지식을 스스로 탐색해가는 방식으로 개발 공부를 해 왔습니다. 필요한 지식을 얻기 위해서 검색하고, 책을 읽고, 공식 문서를 정독하고, 오픈 소스를 뜯어보는 등 다양한 방법을 가리지 않습니다.

Skills

Architecture

  • 저수준 로직과 고수준 로직을 분리해 생각할 수 있습니다.
  • 변화에 유연한 느슨한 결합을 자주 사용합니다.
  • 의존성의 방향에 대해 끊임없이 고민합니다.

JavaScript

  • 최신 ES 표준 문법들을 사용합니다.
  • 비동기 처리(Callback, Promise, async/await)에 대해 잘 이해하고 있습니다.
  • 클로저와 고차 함수를 적극적으로 활용합니다.

TypeScript

  • 인터페이스 활용에 능숙합니다.
  • 다양하고 복잡한 타입들을 큰 어려움 없이 부여할 수 있습니다.
  • any 타입은 쓰지 않습니다. 항상 명확한 타입을 부여합니다.

React

  • 함수형 컴포넌트와 React Hooks를 익숙하게 사용합니다.
  • 커스텀 Hooks를 작성해 복잡한 로직을 모듈화 할 수 있습니다.
  • Storybook을 활용한 컴포넌트 단위 개발을 할 수 있습니다.
  • Recoil을 활용해 전역 상태 관리를 할 수 있습니다.

Server

  • Node.js API를 이해하고 활용할 수 있습니다.
  • Express의 미들웨어 구조에 익숙합니다.
  • SQL을 활용해 데이터를 다룰 수 있습니다.

Security

  • 검증되지 않은 사용자 입력의 위험성을 알고 있습니다.
  • CORS 등의 취약점 방지 기술들을 이해하고 있습니다.
  • 취약점 발생 가능 지점들을 피해 개발하려 항상 노력합니다.
